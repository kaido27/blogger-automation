# Blogger Automation Tool 🚀

This project automates uploading blog posts to Blogger using Python and Google Blogger API.

## Features

* Upload multiple blogs from JSON
* Handles rate limits automatically
* Uploads as drafts

## Setup

1. Install dependencies:

```
pip install -r requirements.txt
```

2. Add your Google API credentials:

* Place `credentials.json` in project folder

3. Run script:

```
python upload.py
```
